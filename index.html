🎼

B站账号同步

高品质播放

优先拉取更高码率音频流，享受清晰饱满的音质，带来沉浸式聆听体验

音频下载

支持一键下载音频文件，可自定义音质设置，离线也能随时收听喜爱的音乐

🧩

精致界面

轻量现代的界面设计，内置浅色与深色主题，搭配毛玻璃效果与流畅动画

💿

系统托盘

支持最小化到系统托盘，后台便捷控制播放，不占用任务栏空间

🎨

多主题配色

提供粉色、紫色、蓝色、绿色、橙色五种主题颜色，随心切换个性界面

📂

本地歌单管理

支持创建、导入、导出本地歌单，JSON格式通用，轻松管理你的音乐库

📺

分P切换支持

完美支持多P视频，可便捷查看并切换分P内容，合集音乐轻松播放

快速上手

简单几步，开启你的音乐之旅

1

搜索与播放

在搜索框输入关键词，回车或点击搜索，点击音乐卡片即可直接播放

2

分P切换

多P视频点击右侧列表按钮，查看所有分P并自由切换播放内容

3

歌单管理

侧边栏创建歌单，支持导入导出JSON格式文件，轻松备份与分享

4

下载音频

音乐卡片或歌单中点击下载按钮，可在设置中调整下载音质

5

界面定制

右上角主题按钮切换颜色主题，支持深色/浅色模式自由切换

6

后台播放

最小化窗口自动隐藏到系统托盘，不影响工作同时享受音乐

源码运行与构建

支持 macOS、Windows、Linux 全平台运行

📦 克隆项目
git clone https://github.com/Rayson-Liu/pink-music-app.git
🔧 安装依赖
npm install
🚀 启动开发服务
npm run dev
🎵 启动 Electron 客户端
npm run electron:dev

构建打包

🍎 macOS
npm run electron:build
🪟 Windows
npm run electron:build -- --win

默认下载目录

🍎 macOS
~/Library/Application Support/Pink Music/downloads/
🪟 Windows
%APPDATA%\Pink Music\downloads\
🐧 Linux
~/.config/Pink Music/downloads/
